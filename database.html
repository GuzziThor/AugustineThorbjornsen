Local Storage and Form Validation

Local storage allows any web application to store data within the browser and retrieve it, even if the browser is closed or restarted.

Local storage typically allows for 5MB of storage space per domain and is primarily used only at the client side.

Local storage also uses a key/value paired system for storing the data.

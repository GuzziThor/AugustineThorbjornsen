About Styles Alive

Hello everyone, today I would like to give you a run down on how I came about choosing my styles, colours and patterns for this website.

In the navigation panel you can read more about style sheets, colour, components and some helpful references.

The main purpose of this site is to give examples of fonts, colours, typographys, tables and show how CSS can be very powerful when used in the right way.



Style

With the overal design of my website being quite simple, I didn't want a fancy design to distract the user from the sites purpose, which was to convey consistancy of colours, styles and patterns that is accessible for everyone.

My design is focused on simplicity and user friendlyness and I hope you feel the same way.



Accessibility Considerations

The colour contrast was edited to suite WCAG AAA standards, while at the same time making it elegant and clean.

Tables contained appropriate caption tags for user understanding. Images contained alt tags in rare case the img doesn't load.

The text is not too small for reading and its colour also meets WCAG AAA standards.

I found my selected fonts from Google Fonts and I loved this site so much I am definitly using it in future web development.
